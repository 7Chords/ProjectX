This is the graduation project repository for Seventh Chord.  
It implements the SCFrame program framework, featuring a turn-based combat system as the TBS component of the framework.  
The framework also supports other functional parts such as UI, Quest, and Stage, along with encapsulated utility modules including an event center, step counter, object pool, state machine, and configuration table tool. It achieves a high degree of decoupling while remaining user-friendly and extensible.  
Several editor extensions have been developed to facilitate rapid creation of game data and functional debugging.  
Game art assets are sourced from free online materials.
